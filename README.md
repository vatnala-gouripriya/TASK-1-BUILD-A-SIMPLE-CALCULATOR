## DESCRIPTION
This project is a basic command-line calculator developed using Python 3.x. It allows users to perform essential arithmetic operations such as addition, subtraction, multiplication, and division. The calculator accepts user input, processes the selected operation, and displays the result. It also incorporates error handling to manage invalid inputs and avoid division-by-zero errors, ensuring reliable program execution.
## OBJECTIVES
The main objective of this project is to create a simple calculator that performs fundamental arithmetic calculations. It helps users solve mathematical problems efficiently while providing practical experience with core Python programming concepts and logic building.
